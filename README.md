# Hearth

## Project setup
```
npm install
```
## Front-end scripts

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

## Back-end scripts

### Start server with auto-reload for development
```
npm run server
```
